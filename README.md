# Hand Gesture Control System

## Overview

The Hand Gesture Control System is a computer vision project that enables touchless interaction through real-time hand gesture recognition. Using OpenCV, MediaPipe, Raspberry Pi, and Firebase, the system detects hand movements and interprets gestures to perform specific actions.

## Features

- Real-time hand tracking and recognition
- Gesture-based control mechanism
- Computer vision implementation using MediaPipe
- Cloud integration using Firebase
- Raspberry Pi deployment

## Technologies Used

- Python
- OpenCV
- MediaPipe
- Raspberry Pi
- Firebase

## Applications

- Smart home control
- Human-computer interaction
- Assistive technologies
- IoT automation

## Future Enhancements
